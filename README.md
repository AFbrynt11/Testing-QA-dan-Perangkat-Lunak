# Testing-QA-dan-Perangkat-Lunak
Project Aplikasi Webiste

  Aplikasi yang dibuat ini adalah aplikasi kependudukan atau dsingkat menjadi APDAK untuk mengelola data penduduk.
Aplikasi ini dirancang menggunakan bahasa pemrograman HTML, CSS, PHP, dan JavaScript dengan 3 fitur utama yaitu:
1. Data Pengguna > admin atau petugas bisa mengetahui siapa saja yang menggunakan website ini.
2. Data Penduduk > admin atau petugas bisa menambah, menghapus, dan mengubah data penduduk bahkan melakukan pencarian terhadap data penduduk.
3. Laporan > admin bisa membuat laporan terkait data pengguna atau penduduk yang sudah ada di kolon keterangan.
aplikasi ini masih sederhana dan kemungkinan akan terus dikembangkan.

- Terkait database yang digunakan berikut isinya:
1. Tabel User > ID (Auto), Username, Password, Role (petugas/penduduk), dan Waktu dibuat (Auto).
2. Tabel Penduduk > ID (Auto), NIK, Nama, Jenis Kelamin, Tanggal Lahir, Tempat Lahir, Alamat, RT, dan RW.
3. Tabel Laporan > ID (Auto), Keterangan, dan Waktu.

- Lalu adanya alur atau Activity dari penggunaan aplikasi kependudukan ini yaitu:
1. Pertama masuk ke halaman awal website atau index berisi ucapan selamat datang dan pengguna diberikan dua pilihan Login atau Register.
2. Jika sudah ada akun langsung ke halaman Login dan jika belum pengguna melakukan Register untuk membuat akun.
3. Di halaman Register ini selain memasukkan Username dan Password adanya button untuk menentukan pengguna ini Petugas atau Penduduk (Role).
4. Setelah membuat akun akan diarahkan ke halaman Login untuk memasukkan akun yang sudah dibuat tadi.
5. Jika Login berhasil akan masuk ke halaman Beranda dan jika gagal akan ada notifikasi "Username/Password/Role Salah!".
6. Setiap pengguna yang sudah masuk akan berada di halaman Beranda yang berisikan jumlah data dari penduduk yang sudah dimasukkan datanya.
7. Lalu adanya menu navigasi yaitu sidebar yang berisikan halaman-halaman yang ada di website tersebut.
8. Dikarenakan adanya Role untuk penggunaan website ini maka akan ada perbedaan tampilkan ketika mengakses website.
9. Admin atau petugas bisa akses semua halaman sementara penduduk hanya bisa akses halaman beranda dan laporan.
10. Setelah selesai menggunakan pengguna bisa melakukan Logout yang ada di menu sidebar.

Catatan: Aplikasi ini masih di tahap pengembangan jadi selama penggunaan website ini masih banyak kekurangannya.

Versi Aplikasi:
1. APDAK 1.0 = Belum menyesuaikan halaman berdasarkan role (petugas/penduduk)
2. APDAK 1.1 = Sudah menyesuaikan halaman berdasarkan role (petugas/penduduk)
